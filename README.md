# WatercolorGIF
### Final term project for NTU Virtual Reality 2018 spring

## Usage:
### Requirements
* python3
* numpy
* Pillow
* imageio
* python-opencv

### Execution
```
python3 watercolor.py [.jpg or .gif]
```
